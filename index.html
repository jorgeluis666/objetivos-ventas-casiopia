Configurar sincronización

Para disparar el workflow de sincronización desde el dashboard, pegá un Personal Access Token de GitHub con permiso workflow. El token se guarda solo en tu navegador (localStorage).

Si no tenés token, la data sigue actualizándose cada lunes vía el cron del workflow.
